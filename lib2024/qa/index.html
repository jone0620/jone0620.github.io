  3. Qus: What trag is used to create the title of a web page? Give an example.

    Ans: The title tag. Example: <title> Content </title>

  4. Qus: What tag is used to create a paragraph in HTML? Give an example.

    Ans: the <p> tag. Example: <p>Paragraph Content </p>

  5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i>. Why?

    Ans: The <i> tag is different, because it is used to italicize text. The others are used as containers for HTML content.

  6. Qus: What are the tags required to create a navigation list in HTML? Give one example with two navigation links.
